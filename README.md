# FinGram: Smart Financial Behaviour Prediction System

## Project Overview
FinGram is a Machine Learning project that analyzes user spending behavior and predicts future expenses.

## Features
- Expense Analysis
- Future Expense Prediction
- Financial Behavior Tracking
- Machine Learning Based Predictions

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## How to Run
1. Download the project.
2. Install required libraries.
3. Run the Python file.
